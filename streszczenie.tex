\vskip 1.2em
\centerline{\bf \emph{Porównanie wydajności serwisów RESTful w językach Java i Go}}
\vskip 1.5em

\setstretch{1.3}

Niniejsza praca przedstawia porównanie wydajności serwisów RESTful w dwóch różnych językach programowania: Java i Go.

Na początku autor zajmuję się omówieniem istniejących metod tworzenia aplikacji internetowych. Opisuje architekturę REST i sposób jej wykorzystania do budowania systemów informatycznych z zastosowaniem mikroserwisów. Omawia też języki Java i Go, które zostały wykorzystane w niniejszej pracy. Przedstawia ich historię oraz różnice między językami w tworzeniu aplikacji internetowych.

Główną część pracy stanowi opis i prezentacja aplikacji stworzonych do przeprowadzenia testów wydajnościowych. Przedstawiona jest ich architektura oraz sposób i przypadki użycia. Następnie autor opisuje testy wydajnościowe, przygotowane i wykorzystane do przeprowadzenia testów. Na koniec, w formie wykresów zaprezentowano wyniki wydajności poszczególnych aplikacji, na podstawie których przeprowadzona jest wnikliwą analizę.
